# Breast-Cancer-Detection
This project focuses on building an intelligent system to classify breast cancer tumors as malignant or benign using supervised machine learning techniques. The model is trained on medical diagnostic data and evaluated using multiple algorithms including Logistic Regression, Random Forest, and Support Vector Machine (SVM).

# Dataset
Dataset: Breast Cancer Dataset (CSV format)
Target Variable: diagnosis
M → Malignant
B → Benign
Features include:
Radius, Texture, Perimeter, Area
Smoothness, Compactness, Concavity, etc.

# Process

Data Cleaning
Exploratory Data Analysis (EDA)
Feature Scaling
Model Training
Model Evaluation

# Models Used

Logistic Regression
Random Forest
Support Vector Machine (SVM)

# Results

Compared multiple models for performance
SVM achieved the highest accuracy 98.9
